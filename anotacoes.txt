Styled Components

- para instalar yarn add styled-components
- para usa-lo criar um arquivo com o estilo colocado dentro de uma tagged template
- fornece alguns recursos do sass, e para referenciar algum componente para hover, focus, usa se o &
- Para usar um componente importado dentro de um styled component basta colocar entre paraenteses o nome do mesmo apos o styled:
    ex: const button = styled(Link)
- Já vem com o child injetado automaticamente
